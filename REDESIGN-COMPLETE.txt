🔄 HOW TO UPDATE
═══════════════════════════════════════════════════════════

In Unraid:
  1. Docker tab → Your container
  2. Force Update
  3. Hard refresh browser (Ctrl+F5)
  4. See new sidebar layout!
  5. Try theme toggle!
  6. Test container controls!

Or rebuild locally:
  cd ~/Documents/GitHub/wordpress/docker-template-manager
  git pull
  docker-compose down
  docker-compose build --no-cache
  docker-compose up -d
